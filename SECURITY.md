# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please open a [GitHub issue](https://github.com/ajustinjames/selling-insert-generator/issues) with the label `security`.

## Scope

This is a fully client-side application with no backend. All data is stored locally in the browser via `localStorage`. No user data is transmitted to any server.

## Supported Versions

Only the latest version on the `main` branch is actively maintained.
